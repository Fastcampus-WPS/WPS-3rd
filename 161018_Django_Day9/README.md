# pip freeze

현재 설치된 파이썬 패키지와 버전정보를 출력  
**pip freeze > requirements.txt**  
입력시 해당 정보를 requirements.txt파일에 저장

**pip install -r requirements.txt**


# 폴더구조는

	- django_app폴더
	- requirements.txt
	- .python-version
